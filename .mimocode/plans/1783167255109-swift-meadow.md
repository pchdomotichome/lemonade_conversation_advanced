# Plan: Lemonade Conversation Advanced - Best Practices Extraction & Roadmap

## Objective

Extract best practices, features, and configurations from 6 reference repos + Lemonade Server API documentation, then create a comprehensive analysis document (`~/workspace/lemonade_conversation_analysis.md`) and implementation roadmap for building the most complete Home Assistant conversation assistant connected to Lemonade Server.

## Phase 1: Documentation Generation

### Step 1: Create `~/workspace/lemonade_conversation_analysis.md`

A single comprehensive markdown file containing:

#### Section A: Repository Analysis (per repo)

For each of the 6 repos, document:
- **Overview**: Stars, license, last activity, primary focus
- **Architecture**: File structure, key patterns, design decisions
- **Feature Inventory**: Complete list of implemented features
- **Best Practices to Adopt**: Specific code patterns, algorithms, or approaches
- **Key Innovations**: What makes this repo unique
- **Code Quality Assessment**: Testing, error handling, documentation
- **License Compatibility**: Whether code can be borrowed safely

#### Section B: Comparative Feature Matrix

Cross-reference table showing which features exist in which repos, and which are missing from Lemonade Conversation Advanced.

#### Section C: Lemonade Server API Capabilities

Document all Lemonade Server endpoints that can be leveraged:
- OpenAI-compatible chat/completions (with tool calling, streaming, vision)
- Embeddings API (for RAG)
- Audio transcription (Whisper) + TTS (Kokoro)
- Model management (pull, load, unload, delete)
- System health/stats
- Realtime WebSocket transcription

#### Section D: Current Codebase Gap Analysis

What exists vs what's missing in `lemonade_conversation_advanced`.

#### Section E: Architecture Design

HA Core + MCP Hybrid architecture:
- Base: HA Core OpenRouter pattern (ChatLog API, subentries, ConversationEntity)
- Add: MCP server for entity discovery (from mcp-assist)
- Add: Lemonade-specific features (model management, hardware sensors)

#### Section F: Implementation Roadmap

Phased roadmap with priorities matching user's selection:
1. MCP entity discovery
2. True streaming + thinking
3. Tool calling in conversation
4. RAG support
5. AI Task entities

#### Section G: New Improvements Analysis

Innovative features not in any analyzed repo that could make this the most complete assistant.
