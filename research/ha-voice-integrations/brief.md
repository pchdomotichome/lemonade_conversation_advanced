# Research Brief: Home Assistant Voice Assistant Integrations

## Refined question
What is the current landscape of Home Assistant voice assistant integrations — built-in features, popular custom integrations, modern architecture patterns, configuration UI conventions, and essential user features?

## Scope boundaries
- **In**: Official HA voice pipeline, HACS custom integrations, config flow patterns, modern LLM-based approaches, key user expectations
- **Out**: Home Assistant in general (focus on voice only), commercial smart speakers beyond HA integration, competing non-HA platforms
- **Time frame**: 2024-2026 (current state, recent developments)

## Assumptions
- Reader is a developer building HA custom integrations
- Focus on technical architecture, not marketing
- Include GitHub URLs for all key projects

Standard (3-5 sub-agents, ~20 sources)
